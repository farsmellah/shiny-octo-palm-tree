# Laravel Tasks API

## Стек

-   Laravel
-   MySQL
-   PHP 8.4
-   Docker

## Быстрый старт

```bash
make quickstart
```

## Эндпоинты

```bash
GET    /api/tasks          # Список задач
POST   /api/tasks          # Создать задачу
GET    /api/tasks/{id}     # Получить по id
PUT    /api/tasks/{id}     # Обновить
DELETE /api/tasks/{id}     # Удалить
```
